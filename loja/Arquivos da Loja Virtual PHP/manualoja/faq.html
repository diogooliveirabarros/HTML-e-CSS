MANUAL DE INSTALAÇÃO E OPERAÇÃO  DA LOJA VIRTUAL OSCOMMERCE 2.2 - VERSÃO PORTUGUÊS

Notas

1) Este Sistema de Loja Virtual em PHP é Open Source, ou seja, código fonte aberto. Isto significa que cada desenvolvedor poderá alterar livremente o código fonte da loja de acordo com suas necessidades. É importante salientar que para isto, será necessário um conhecimento básico da linguagem PHP. Esta loja é um software livre. Portanto pode ser distribuída e utilizada livremente!

2) Infelizmente não temos condições de oferecer suporte às dúvidas de  cada usuário. Foi justamente por este motivo que este manual foi criado, com a finalidade de orientar o usuário final na instalação e alteração de sua Loja Virtual. Como já foi mencionado anteriormente, é muito importante ter um prévio conhecimento da linguagem PHP para trabalhar com este aplicativo. Inserimos neste CD-ROM uma apostila de PHP como material de apoio para que os leigos possam se tornar aptos a trabalhar com PHP. Neste caso, vale o esforço e a dedicação de cada um para o aprendizado!

Requerimentos básicos

Para que possa configurar a loja virtual 2.2 br, os sequintes pré requesitos são necessários.

- Um Computador com um sitema instalado e funcionando convenientemente para as configurações da loja virtual

Um Web Server com:

- Servidor Apache (http://www.apache.org)
- PHP versão 4.1.x ou melhor (mod_php), módulo do Apache (http://www.php.net)
- database MySQL (http://www.mysql.com)
- E com no mínimo 20 mgs de espaço e com endereço de url

OBS: Pode se utilizar pacotes instalaveis que emulam servers para testes locamente como: Easyphp, Phpdev, AppServ entre outros.
Mais sua utilização não promete 100% de compatibilidade.

Configurações e instalação

- Crie o banco de dados em seu server sql/phpmyadmin onde vai ficar o banco de dados da loja virtual

- Configure os arquivos configure.php que estão dentro das pastas:

- comercio/includes/configure.php
- comercio/admin/includes/configure.php

de acordo com seu servidor (site). Abaixo a variaveis de configuração.

// Define seu webserver com sua URL
define('HTTP_SERVER', 'http://www.seusite.com'); // seu site URL
define('HTTPS_SERVER', 'https://www.seusite.com.br'); // seu site URL

// define sua database mysql conecção
define('DB_SERVER', 'localhost'); // local da sua database server
define('DB_SERVER_USERNAME', 'database_name'); // nome usado na database
define('DB_SERVER_PASSWORD', '12345'); // password usado na database
define('DB_DATABASE', 'database_name'); // nome da database

define('DIR_FS_DOCUMENT_ROOT', '/home/seusite/public_html'); // caminho do server, OBS: somente no admin/includes

- Após ter feito as configurações, envie a pasta comércio com todo o conteúdo para seu web site mantendo a estrutura.

- Importe para o seu já criado banco de dados sql via cpanel/sql/phpmyadmin do seu server, o bd oscommerce.sql que esta dentro da pasta SQL. Ele é a base de dados da loja virtual.

- Feito isto tente acessar a sua loja virtual indo em: www.seusite.com.br/comercio, se abrir a página inicial é porque tudo foi corretamente instalado.

Erro. Se der este erro: = Unable to connect to database server! =, deve ser porque não foi configurado corretamente os configure.php, revise estes passos.

Permissões (CHMOD) em arquivos e pastas

- Dê as seguintes permissões (Chmod) nas pastas e arquivos, conectado em seu server.
- Para fazer isto basta se conectar com o ftp (Cuteftp, Smartftp ou outro) e clicar com o botão do mouse no arquivo e escolher chmod.


- comercio/admin/backups ... 777 // para os backups do bd
- comercio/images ... 777 // para poder guardar imagens
- comercio/stylesheet ... 777 // para modificar cores da loja, caso não for modificar deixe em 755

- o resto das pastas e arquivos não precisa de permissões diferentes.

Protegendo o admin da loja virtual com senha

OBS: Nem todos cpanel são iguais, tem algumas diferenças nos icones e textos.
- Para a segurança da sua Loja Virtual, recomendamos que você proteja o seu Administrador com um Login e Senha, cujo processo de inserção será mostrado aqui: Entre no CPANEL (Painel de Controle) do seu domínio: http://www.seudominio.com.br/cpanel, utilize o login e senha que foi lhe fornecido em um e-mail de retorno após a abertura de sua conta.


- Após ter se logado, você entrará na primeira página do CPANEL, nessa página procure pelo item "Proteger Diretórios".


- Agora você só precisa localizar a pasta "admin". Para isso clique na pasta "comércio" e aguarde a tela se abrir, pronto. A pasta "admin" estará dentro da pasta "comércio"


- Quando você localizar a pasta "admin", clique no no nome "admin" e não na pasta. Isso fará com que você acesse as propriedades dessa pasta e possibilitará você de colocar a senha no mesmo.


- Se abrirá uma nova página, nela você terá que inserir primeiro o "Nome do recurso" e clique em "Gravar".


- Feito isso você só precisará inserir o seu login e senha para acesso a pasta "admin", ou seja ao Administrador da Loja Virtual e clicar no botão "Adicionar novo usuário"

Imagens

- As imagens dos produtos ficam na pasta comercio/images. Tenha em mente trabalhar com imagens no tamanho pequeno de 100x100, não ultrapassando este tamanho em altura ou largura. Ista fará com que a imagem não fique destorcida.


- Foi craida uma pasta dentro da pasta comercio chamada html onde vai ficar as imagens de tamanhos grandes. Isto melhora a qualidade e pode-se adicionar quantas imagens de um mesmo produto e com mais informações em texto. Crie as páginas em html com as imagens dos produtos e textos e envie para a pasta html. Assim quando se clicar para ver mais sobre o produto abre a página pop-up criada para isto.

Largura da imagem pequena: Largura que você deseja para as imagens dos produtos de sua loja pode ser em pixels(mais comum) ou centímetros, se for usar padrão 100x100 deixe em branco.
Altura da imagem pequena: Altura que você deseja para as imagens dos produtos de sua loja pode ser em pixels(mais comum) ou centímetros, se for usar padrão 100x100 deixe em branco.
Largura da imagem do cabeçalho: Largura que você deseja para as imagens dos cabeçalhos de sua loja pode ser em pixels(mais comum) ou centímetros, se for usar padrão 100x100 deixe em branco.
Altura da imagem do cabeçalho: Altura que você deseja para as imagens dos cabeçalhos de sua loja pode ser em pixels(mais comum) ou centímetros, se for usar padrão 100x100 deixe em branco.
Largura da imagem da subcategoria: Largura que você deseja para as imagens das categorias e subcategorias – pixels e centímetros, se for usar padrão 100x100 deixe em branco.
Altura da imagem da subcategoria: Altura que você deseja para as imagens das categorias e subcategorias – pixels e centímetros, se for usar padrão 100x100 deixe em branco.
Calcular tamanho da imagem: preferencialmente deixe (true)
Imagem Obrigatória: se você deseja que as imagens sejam obrigatórias deixe (true), se não desejar deixe (false) padrão.

Ferramentas

Cópia de segurança: É de extrema importância que você mantenha sempre uma cópia de segurança de sua loja. Clique em “backup”, selecione o compactador de sua preferência, (não use compactar pode dar erros). Tente manter uma cópia no server para uma emergêcia. Ela fica na pasta admin/backup, e tente sempre manter atualizado isto.


Editar textos: Por aqui você pode configurar todos os textos de sua loja. Já vem pré-configurado, porém se desejar, basta modifica-los. Neste caso você tem que dar permissão de escrtita (CHMOD) de 777 em todos os arquivos para que seja modificado por aqui. Veja em Linguagens mais acima uma maneira mais fácil de editat os textos.
